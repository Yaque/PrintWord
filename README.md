# PrintWord
打字练习平台

当前版本是使用python3+flask+jquery框架搭建
## 实例图片
![实例图片](./r.png)

## 使用
1.首先使用create_sqlite.py生成sqlite数据库
2.然后正常运行即可

具体步骤：

    pip install Flask
    python3 create_sqlite.py
    python3 app.py
    
默认可以在局域网内访问。

## 平台支持
1.理论上支持任何可运行flask的平台

## 建议
1.服务器部署建议使用nginx
